Cum lucrăm

How we work

Metodologie clară, fără surprize

Clear methodology, no surprises

Fiecare angajament urmează același proces: scoping, execuție, raport tehnic + executiv, re-test după remediere. Știți exact ce se întâmplă la fiecare pas.

Every engagement follows the same process: scoping, execution, technical + executive report, re-test after remediation. You know exactly what happens at each step.

01

Black Box

Black Box

Intrăm fără nicio informație prealabilă despre sisteme — exact cum ar face un atacator real de pe internet. Cel mai realist scenariu.

We go in with zero prior information about the systems — exactly as a real attacker from the internet would. The most realistic scenario.

02

Grey Box

Grey Box

Testăm cu credențiale de utilizator normal sau informații parțiale. Ideal pentru scenariul angajat compromis sau cont spart.

We test with normal user credentials or partial information. Ideal for the compromised employee or hacked account scenario.

03

White Box

White Box

Acces complet la cod sursă, arhitectură, documentație. Cel mai eficient pentru code review și audit profund înainte de lansare.

Full access to source code, architecture, documentation. Most efficient for code review and deep audit before launch.

04

Scoping gratuit

Free scoping

Înainte de orice ofertă, discutăm ce aveți, ce vreți să testăm și cât de adânc. Fără presiune, fără angajament.

Before any quote, we talk about what you have, what you want tested and how deep. No pressure, no commitment.

05

Execuție & documentare

Execution & documentation

Testare manuală cu proof-of-concept pentru fiecare vulnerabilitate găsită. Nu liste automate de CVE-uri.

Manual testing with proof-of-concept for every vulnerability found. Not automated CVE lists.

06

Raport + re-test inclus

Report + re-test included

Raport executiv (ce risc, ce impact) + tehnic (cum să repari). Re-testare gratuită după ce ați remediat.

Executive report (what risk, what impact) + technical (how to fix). Free re-test after you've remediated.
